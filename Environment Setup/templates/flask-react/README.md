# Flask Environment

Flask + TypeScript Stack project initialized by Scripty

## Project Structure
(folder_name)/
- backend/         # Flask Python backend
- frontend/        # React TypeScript frontend

## Getting Started

1. Start the backend:
   cd backend
   # If using Poetry:
   poetry install
   poetry run flask run
   # If using pip:
   pip install -r requirements.txt
   flask run

2. Start the frontend:
   cd frontend
   npm install
   npm run dev
